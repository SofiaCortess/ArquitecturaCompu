Unidad 3: Selección de componentes para ensamble de equipo de cómputo

3.1 Chipset
3.1.1 Unidad central de procesamiento (CPU)

Factores clave para selección:

  • Socket: Compatibilidad con placa madre (LGA, PGA, BGA)
  • Núcleos/Hilos: 4/8, 6/12, 8/16 para diferentes cargas de trabajo
  • Frecuencia: GHz base/boost (3.5GHz/4.7GHz)
  • TDP: Consumo térmico (35W, 65W, 105W+)
  • Generación: Intel Core i9-13xxx, AMD Ryzen 5000/7000
  • Gráficos integrados: Intel UHD, AMD Radeon Graphics
3.1.2 Controlador de Bus

Componente que gestiona la comunicación entre dispositivos:

  • Tipos de buses: FSB, DMI, HyperTransport, Infinity Fabric
  • Ancho de banda: Determinado por frecuencia y bits (64-bit @ 100MHz)
  • Arbitraje: Priorización de dispositivos
  • Chipsets modernos: Intel Z790, AMD X670
3.1.3 Puertos de entrada/salida

Interfaces para conectar periféricos:

  • USB: 2.0 (480Mbps), 3.2 Gen 2x2 (20Gbps), USB4 (40Gbps)
  • Thunderbolt: 3/4 (40Gbps), compatible USB4
  • Video: HDMI 2.1, DisplayPort 2.0
  • Audio: Jack 3.5mm, S/PDIF óptico
  • Red: Ethernet 2.5G/10G, Wi-Fi 6E
3.1.4 Controlador de acceso directo a memoria

Gestión de transferencias sin CPU:

  • Modos DMA: Single, Block, Demand, Cascade
  • Canales DMA: 8 canales en sistemas modernos
  • Velocidades: Hasta 6.4GT/s (PCIe 4.0)
  • Optimizaciones: Scatter-Gather, Chaining
3.1.5 Controlador de interrupciones

Manejo de señales asíncronas:

  • Controladores PIC/APIC: 8259A vs I/O APIC moderno
  • Prioridades: Niveles IRQ 0-15 (IRQ0 timer, IRQ1 keyboard)
  • MSI: Message Signaled Interrupts (PCIe)
  • Niveles: Edge-triggered vs level-triggered
3.1.6 Circuitos de temporización

Sincronización de operaciones:

  • Reloj del sistema: Base de tiempo (14.318MHz crystal)
  • PLL: Phase-Locked Loops para multiplicar frecuencia
  • Timers: 8253/8254 Programmable Interval Timer
  • RTC: Real-Time Clock con batería CMOS
3.1.7 Circuitos de control

Gestión de flujo de operaciones:

  • Reset circuits: Power-on reset, watchdog timer
  • Voltage regulation: VRMs para CPU/GPU (fases 8+2, 10+2)
  • Power management: ACPI estados (S0-S5)
  • Fan control: PWM controlado por temperatura
3.1.8 Controladores de video

Gestión de salida gráfica:

  • GPU integrada: Intel UHD, AMD Radeon Graphics
  • Salidas: Soporte para múltiples monitores
  • Aceleración: Quick Sync Video, NVENC
  • Resoluciones: Hasta 8K @ 60Hz (DP 2.0)
  • API soportadas: DirectX 12, Vulkan, OpenGL
3.2 Aplicaciones
3.2.1 Entrada/Salida

Selección de dispositivos periféricos:

  • Teclados: Mecánicos vs membrana, switches (Cherry MX)
  • Ratones: Óptico vs láser, DPI ajustable (800-16000+)
  • Monitores: TN vs IPS vs VA, Hz (60-360Hz), resolución (1080p-8K)
  • Impresoras: Laser (PPM), inyección (DPI), multifuncionales
3.2.2 Almacenamiento

Opciones de almacenamiento modernas:

  • SSD NVMe: PCIe 3.0/4.0 (3500-7000MB/s), formatos M.2 2280
  • SSD SATA: 2.5" (550MB/s), para almacenamiento secundario
  • HDD: 3.5" (7200RPM, 256MB cache), para archivado
  • NAS: Soluciones multi-bay para RAID (Synology, QNAP)
  • Cloud: Backup automático (OneDrive, Google Drive)
3.3 Ambientes de servicio
3.3.1 Negocios

Requisitos para entornos corporativos:

  • Estaciones de trabajo: Dell OptiPlex, Lenovo ThinkCentre
  • Portátiles empresariales: Durabilidad MIL-STD, TPM
  • Servidores: Torre/rack (PowerEdge, ProLiant)
  • Seguridad: BitLocker, vPro, SmartCard readers
  • Gestión: Intel AMT, LANDesk
3.3.2 Industria

Equipos para entornos industriales:

  • Rugged PCs: Fanless, -20°C a 60°C operación
  • Panel PCs: IP65, resistentes a químicos
  • PLC: Controladores lógicos programables
  • Adquisición de datos: Módulos DAQ PCI/PCIe
  • Certificaciones: ATEX, IEC 61131-2
3.3.3 Comercio Electrónico

Infraestructura para e-commerce:

  • Servidores web: Escalabilidad vertical/horizontal
  • Balanceo de carga: HAProxy, F5 BIG-IP
  • Bases de datos: MySQL Cluster, MongoDB Atlas
  • CDN: Cloudflare, Akamai para contenido estático
  • Seguridad: WAF, DDoS protection
